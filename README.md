# 신시도 날씨

신시도 지역의 실시간 날씨 정보를 제공하는 웹사이트입니다.
기상청 공공데이터 포털의 단기예보 API를 사용합니다.

## 사용 방법

1. 기상청 공공데이터 포털에서 인증키를 발급받으세요.
   - https://www.data.go.kr 접속
   - 회원가입 후 "기상청_단기예보 ((구)_동네예보) 조회서비스" 검색
   - 활용신청 후 인증키 발급

2. `script.js` 파일을 열어서 `YOUR_API_KEY_HERE` 부분을 발급받은 인증키로 변경하세요.

3. `index.html` 파일을 브라우저에서 열어서 날씨 정보를 확인하세요.

## 좌표 정보

- 위치: 신시도
- 위도: 37.290278° (북위 37° 17′ 25″)
- 경도: 126.427778° (동경 126° 25′ 40″)

## 기능

- 현재 온도 및 날씨 상태 (기상청 초단기실황)
- 체감온도 (기온으로 표시)
- 습도
- 풍속
- 10분마다 자동 업데이트

## 주의사항

- 기상청 API는 CORS 정책으로 인해 브라우저에서 직접 호출 시 오류가 발생할 수 있습니다.
- CORS 오류가 발생하는 경우, 프록시 서버를 사용하거나 백엔드 서버를 통해 API를 호출해야 합니다.
